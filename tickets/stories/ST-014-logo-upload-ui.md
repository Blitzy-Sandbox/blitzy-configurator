## Narrative

As an end user, I want to upload my own logo image through the configurator, so that I can brand the ball with my own visual identity.

## Acceptance Criteria

- [x] The control sidebar exposes a file-picker affordance that accepts standard raster and vector image formats for logo upload.
- [x] A valid uploaded logo appears on the preview ball without requiring a page reload.
- [x] The uploaded logo persists on the preview across rotation, color, pattern, and finish changes for the lifetime of the current design.
- [x] The file-picker is reachable and operable using only keyboard input and is labeled for assistive technology.
