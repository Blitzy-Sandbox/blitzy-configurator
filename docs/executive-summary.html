PART 4

Onboarding

From clean clone to running stack in six commands.

Local quick-start

Local Development Quick-Start

  1. Run nvm use (auto-selects Node 20 from .nvmrc).
  2. Run npm install at the repository root.
  3. Run docker compose up -d to start PostgreSQL, Firebase Auth emulator, and the GCS emulator.
  4. Run docker compose exec backend npx node-pg-migrate up to apply the three schema migrations.
  5. Run npm run dev (frontend on :5173, backend on :3000).
  6. Verify curl localhost:3000/healthz returns {"status":"ok"}.
